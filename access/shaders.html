当サーバーはMinecraft Java Edititon版のOptifine 1.18.2を起動構成に使用したプラグインサーバーとなります。
内容はファンタジーのMMORPGです。

推奨影Modの導入(任意)

導入方法

当サーバーでは、夜の時の暗さなどの関係で比較的暗すぎない影modを推奨影modとして導入をお願いしています

スペックが足りない場合や環境的に入らない場合など無理に入れる必要はございません

サーバーをよりリアルな画質で楽しみたい方は是非このページの導入方法を見て影modを入れてみてください

①リンクにアクセスする

https://modrinth.com/shader/bsl-shaders/version/8.2.0

②BSL v8.2.08と書かれた場所の下にある 緑色のDownloadボタンを押す※.zip形式のまま使用するので解凍しないこと

③起動構成が「Optifine1.18.2」であることを確認した後「プレイ」を押して起動する※初回起動の方は言語を日本語に変えてください

④トップのメニュー画面から設定⇒ビデオ設定⇒シェーダーの詳細設定を開き左下のシェーダーフォルダーのボタンを押すとエクスプローラーが開かれるので
そのフォルダに先ほどダウンロードした「BSL v8.2.08.zip」を圧縮されたままの形式でフォルダーに配置する
